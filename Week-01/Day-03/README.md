# 📚 Topics Covered

## 1️⃣ LEFT JOIN

* Retrieving all records from the left table
* Displaying unmatched rows as `NULL`

## 2️⃣ RIGHT JOIN / Equivalent LEFT JOIN

* Fetching all records from the right-side table
* Understanding database compatibility limitations

## 3️⃣ FULL OUTER JOIN / UNION Simulation

* Combining unmatched rows from both tables
* Simulating full joins using `UNION`

## 4️⃣ INNER JOIN Concepts

* Matching records between related tables

## 5️⃣ NULL Filtering

* Identifying missing enrollments
* Detecting courses without instructors
* Finding unmatched relationships

## 6️⃣ Multi-Table Joins

* Joining Students, Enrollments, Courses, and Instructors
* Building structured reports

## 7️⃣ Relationship Analysis Queries

* Students not enrolled in courses
* Courses without enrollments
* Courses without instructors
* Instructor-course assignments

## 8️⃣ CROSS JOIN

* Generating every possible student-course combination

## 9️⃣ Data Integrity Analysis

* Verifying relationship consistency
* Identifying orphaned records

## 🔟 Join Optimization Concepts

* Choosing correct join types
* Query readability and efficiency

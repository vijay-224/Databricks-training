# 📊 SQL Window Functions and CTE Practice

## 🎯 Objective

The objective of this practice session is to understand and implement advanced SQL analytical operations using relational database tables such as **Employees**, **Customers**, and **Orders** with **Window Functions** and **Common Table Expressions (CTEs)**.

---

# 🧱 Database Setup

The following tables were created and used for SQL analytical query practice:

* 👨‍💼 Employees
* 🧑‍💼 Customers
* 🛒 Orders

The database schema includes:

* Primary Key constraints
* Foreign Key relationships
* Employee-manager hierarchy
* Salary and sales transaction records
* Sample data insertion for analytical reporting

---

# 🔑 Core Concepts Implemented

* Primary Keys
* Foreign Keys
* Relational Mapping
* Data Insertion
* Window Functions
* Common Table Expressions (CTEs)
* Recursive Queries
* Analytical Reporting Operations
